# AI Sales Coach

A real-time AI Sales Coach application featuring a robust **Video and Audio Calling Facility**. This project provides a seamless, Google Meet-style conferencing experience tailored for sales professionals.

## 🚀 Key Features

### 🎥 Complete Audio & Video Calling Facility
- **High-Quality Conferencing**: Fully functional audio and video calling capabilities powered by Stream SDK.
- **Google Meet-Inspired UI**: A professional, familiar interface with Grid and Speaker layouts.
- **Device-Smart Camera Flip**: Automatically detects and offers "Flip Camera" options on mobile/tablet devices with multiple cameras.
- **Mobile Optimized**: Responsive design with touch-friendly controls and centered popups.

### 🛠️ Interactive Controls
- **Smart Grid**: Automatically resizes participants to fill the screen (Fit/Fill view options).
- **Screen Sharing**: One-click screen sharing with visual indicators.
- **Integrated Sidebar**: Access participant lists and meeting info without leaving the call.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Next.js 14+** (App Router)
- **TypeScript**
- **Stream Video SDK**
- **Tailwind CSS**
- **Shadcn/UI**
